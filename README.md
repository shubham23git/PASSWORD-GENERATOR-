# Password Generator App

## Description
A React-based password generator application that creates secure and customizable passwords in real time based on user-selected options such as length, symbols, numbers, and uppercase characters.

## Features
- Generate strong random passwords
- Customize password length
- Include/exclude numbers, symbols, and uppercase letters
- Copy password to clipboard
- Responsive UI

## Tech Stack
- React
- Vite
- JavaScript
- CSS
- tailwind
